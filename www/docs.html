Getting Started

Prerequisite

  • Download Java 1.4+ (Java 5.0 recommended)
  • Download Ant and add Ant's bin directory to your system PATH.
Checkout source code
  • cvs -d :pserver:yourid@cvs.dev.java.net:/cvs login
  • cvs -d :pserver:yourid@cvs.dev.java.net:/cvs checkout testplayer
Building TestPlayer
  • Open Shell and type:
  • cd TestPlayer
  • ant
Testing TestPlayer
  • Open Shell and type:
  • cd samples/mini
  • ant run
  • View uml and test folders uner mini/src/

Library Dependencies

  1. JUnit 3.8.1 and JUnit 4.0 for all unit tests.
  2. TestNG 4.5.2 for all unit tests.
  3. Assertion Extension for JUnit
  4. jMock 1.0.1 for mocking collaborating services.
  5. CGLib 2.1.3 for mocking concrete classes.
  6. AspectJ 1.5 for aspect oriented technology.
  7. AspectWerkz 2 for aspect oriented technology.
  8. Fit and FitNesse for Functional Testing.
  9. Sequence library for generating sequence diagrams.
  10. Download Dot Graphics for generating association and package dependencies diagrams.
  11. Ant 1.6.3, Ant-Contrib, Maven 2.0
  12. XStream 1.1.3 for marshalling object states.
  13. Coverage and quality metrices such as Cobertura, PMD, CheckStyle, FindBugs, JDepend, etc.
  14. JUnitPerf to measure performance of unit tests
  15. Grinder to run load testing.
